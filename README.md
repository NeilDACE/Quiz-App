# The Unofficial Wizarding World Knowledge Test

Welcome to the **Unofficial Wizarding World Knowledge Test** – a simple, interactive fan quiz built using standard web technologies (HTML, CSS, JavaScript).

This project is a tribute to the magical world created by J.K. Rowling and allows fans to challenge their knowledge on a variety of topics, from Hogwarts trivia and spells to magical creatures and characters.

## ✨ Features

* **40 Unique Questions:** A pre-filled set of 40 unique questions based on the magical universe.
* **Simple UI:** Clean, responsive design for an engaging user experience.
* **Score Tracking:** Tracks the player's correct answers throughout the quiz session.
* **Fan-Art Assets:** Features magically-themed background art and icons created to minimize copyright risk.
